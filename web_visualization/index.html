Welcome!

This application visualizes post data from the Social TUI application.

To get started:

  1. Run the websocket server: python websocket_server.py
  2. Run the TUI with websocket support: ./interactive_posts.py --websocket-port 8765
  3. Select a post in the TUI to see the details here.
